# Junior-Farm 🫐

> A modern, premium e-commerce web application for Junior's Farm based in Dehradun. Built using React, Vite, and Tailwind CSS.

## 🌟 Overview

Junior-Farm is the official frontend web application designed to showcase and sell premium, farm-fresh blueberries directly from Dehradun. The platform features a responsive, sleek, and modern user interface targeted toward a conversion-focused user experience.

## 🚀 Features

- **Dynamic, Interactive UI:** A modern design leveraging clean aesthetics and seamless block-level animations.
- **Product Storefront:** Beautifully crafted product cards and sections highlighting premium blueberries and packaging.
- **Streamlined Ordering:** A fluid `OrderModal` component that makes the checkout process straightforward.
- **Fully Responsive Layout:** A flawless experience across devices—mobile, tablet, or desktop.
- **Component-Driven Architecture:** Clean, modular React codebase (Hero, About, Contact, Footer, Products, etc.).

## 🛠️ Tech Stack

- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/) for quick development and optimized production builds.
- **Language:** JavaScript (`.jsx`)
- **Styling:** Vanilla CSS & [Tailwind CSS](https://tailwindcss.com/) for highly customizable, utility-first design.

## 💻 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/juniors-farm.git
   cd juniors-farm/client
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the Application:**
   Open your browser and navigate to the local server, typically `http://localhost:5173`.

## 🎨 Design & Aesthetics
A strong emphasis has been placed on user experience, adopting a vibrant, minimal layout that complements the premium nature of the blueberry brand. The project extensively leverages modern typography, curated color palettes, and glassmorphism styling queues to "wow" the user at first glance.
